# Quy chuẩn thuật ngữ chứng khoán tiếng Việt (VN Securities Glossary)

> Dùng cho báo cáo phân tích cổ phiếu Việt Nam. Mục tiêu: **luôn tiếng Việt chuẩn ngành**
> (như CafeF / VietStock / SSI / HOSE dùng), **tuyệt đối không tiếng Trung**, tránh dịch
> máy móc nghe "ngang". Bảng này là nguồn chân lý: dùng cho prompt gửi Gemini và để dịch
> nhãn báo cáo.

## 1. Khuyến nghị hành động (decision actions)
| Khái niệm gốc | ❌ Dịch ngang | ✅ Chuẩn Việt |
|---|---|---|
| strong_buy / 强烈买入 | Mua mạnh mẽ | **Mua mạnh** |
| buy / 买入 | | **Mua** |
| hold / 持有 | | **Nắm giữ** |
| watch / 观望 | Quan vọng | **Quan sát / Đứng ngoài** |
| reduce / 减仓 | Giảm kho | **Giảm tỷ trọng** |
| sell / 卖出 | | **Bán** |
| strong_sell / 强烈卖出 | | **Bán mạnh** |

## 2. Xu hướng & tín hiệu kỹ thuật
| Gốc | ✅ Chuẩn Việt |
|---|---|
| 看多 / bullish | Tích cực / Thiên mua |
| 看空 / bearish | Tiêu cực / Thiên bán |
| 震荡 / sideways | Đi ngang / Tích lũy |
| 强烈看多/看空 | Rất tích cực / Rất tiêu cực |
| 多头排列 (MA5>MA10>MA20) | **MA xếp tăng** (MA5>MA10>MA20) |
| 空头排列 (MA5<MA10<MA20) | **MA xếp giảm** (MA5<MA10<MA20) |
| 均线 | Đường trung bình động (MA) |
| 金叉 / 死叉 | **Cắt lên** (golden cross) / **Cắt xuống** (death cross) |
| 超买 / 超卖 | Quá mua / Quá bán |
| 乖离率 | Độ lệch MA (BIAS) |
| 趋势强度 | Độ mạnh xu hướng |
| 支撑位 / 压力位 | **Ngưỡng hỗ trợ / Ngưỡng kháng cự** |
| 振幅 | Biên độ dao động |
| 量比 | Tỷ lệ khối lượng (so với TB) |
| 换手率 | **Thanh khoản / Tỷ lệ khớp lệnh** (không dịch "tỷ lệ chuyển nhượng") |
| RSI / MACD / ADX / Bollinger / ATR | Giữ nguyên (thuật ngữ quốc tế) |

## 3. Giá & giao dịch
| Gốc | ✅ Chuẩn Việt |
|---|---|
| 收盘 / 开盘 / 最高 / 最低 | Đóng cửa / Mở cửa / Cao nhất / Thấp nhất |
| 昨收 | **Tham chiếu** (giá đóng cửa phiên trước) |
| 涨跌幅 / 涨跌额 | % thay đổi / Mức thay đổi |
| 涨停 / 跌停 | **Trần / Sàn** (HOSE ±7%, HNX ±10%, UPCOM ±15%) |
| 成交量 (股) | **Khối lượng (KL)** — đơn vị: cổ phiếu (cp) |
| 成交额 (元) | **Giá trị giao dịch (GTGD)** — đơn vị: VND |
| T+1 (TQ) | **T+2** (VN) |

## 7. Vị thế & quản trị rủi ro
| Gốc | ✅ Chuẩn Việt |
|---|---|
| 持仓者 / 空仓者 | Người đang nắm giữ / Người chưa có vị thế |
| 仓位 | **Tỷ trọng / Vị thế** |
| 建仓 / 加仓 / 减仓 | Giải ngân / Gia tăng / Giảm tỷ trọng |
| 止损位 | **Ngưỡng cắt lỗ** (stop-loss) |

| 风控 | Quản trị rủi ro |

## 8. Quy tắc chung cho LLM
- Viết **100% tiếng Việt**, không chèn chữ Hán, không pinyin.
- Số tiền: VND, dùng "tỷ"/"nghìn tỷ" (vd "120,6 nghìn tỷ VND"), không dùng "万亿/亿".
- Đơn vị cổ phiếu viết "cp" hoặc "cổ phiếu"; khối lượng "triệu cp".
- Ngày: định dạng dd/mm/yyyy.
- Không áp khái niệm A-share (trần ±10%, 北向, 龙虎榜, T+1) — dùng đúng VN (HOSE ±7%, khối ngoại, T+2).
- Giữ nguyên thuật ngữ quốc tế đã phổ biến: RSI, MACD, ADX, Bollinger, P/E, P/B, ROE, ROA, MA.
